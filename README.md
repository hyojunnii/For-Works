## 3. 프로젝트 소개
기업 구성원들을 위한 협업툴입니다.  <br>
기업에 필요한 구성원과 업무를 위한 기능들을 한 곳에서 모두 지원하여 편의성을 제공하고, 업무를 공유하고 관리할 수 있는 공간을 만들어 업무의 효율성을 높이며 기업 업무에 대한 이해도를 높이고자 기획하였습니다. <br>
네이버웍스, 다우오피스, 시프티 등을 참고하였습니다.

### 4.1. 팀원별 기능
